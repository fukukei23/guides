tmux チートシート

基本概念

概念説明
セッションtmuxの最大単位。名前付きで複数保持可能
ウィンドウセッション内のタブ。1セッションに複数
ペインウィンドウ内の分割画面
プレフィックスキー: Ctrl+B（以下 C-b と表記）

セッション操作

やることコマンド
新規セッション作成
tmux new -s 名前
セッション一覧
tmux ls
セッションに接続
tmux attach -t 名前
セッション終了
tmux kill-session -t 名前
全セッション終了
tmux kill-server

デタッチ・アタッチ

やること操作
デタッチ（抜ける、セッション維持）C-b → D
アタッチ（戻る）
tmux attach -t 名前
他端末から同じセッションに参加
tmux attach -t 名前
SSH接続が切れてもtmuxセッションは生きています。再接続して attach すれば続きから再開

ウィンドウ操作

やること操作
新規ウィンドウC-b → C
次のウィンドウC-b → N
前のウィンドウC-b → P
ウィンドウ番号で移動C-b → 0-9
ウィンドウ名変更C-b → ,
ウィンドウ閉じるC-b → &

ペイン操作（画面分割）

やること操作
左右分割C-b → %
上下分割C-b → "
ペイン間移動C-b → 矢印キー
ペイン終了C-b → X または exit
ペインサイズ変更C-b → Ctrl+矢印
レイアウト切替C-b → Space

よく使う組み合わせ

Claude Codeを常時稼働
tmux new -s claude && claude
SSH経由で再接続
tmux attach -t claude
← ガイド一覧に戻る
