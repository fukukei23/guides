よく使うコマンド Top 15

コマンド説明
/compact会話を要約してコンテキスト節約 必須
/modelモデル切り替え（Sonnet/Opus/Haiku等）
/cost現在のセッションのトークン使用量・料金
/clear会話履歴をクリア（新規開始）
/plan実行前に計画を立てるモードに入る
/code-reviewコードのバグ・品質をレビュー 便利
/reviewPRをローカルでレビュー
/diff変更差分を対話的に確認
/memoryCLAUDE.md・メモリファイルを編集
/contextコンテキスト使用量をビジュアル表示
/loop同じプロンプトを繰り返し実行（自動化）
/runアプリを起動して動作確認
/verifyビルド＋実行で変更が動くか確認
/batch大規模な変更を並列実行
/fast高速モード切替（Opus高速出力）

内蔵スキル一覧

スキルは /skill名 で呼び出せる事前定義のワークフローです。

スキル説明
/code-reviewコードレビュー（バグ・品質チェック）
/batch大規模変更の並列オーケストレーション
/runアプリ起動＋動作確認
/verifyビルド＋テストで変更検証
/loopプロンプトの定期実行
/debugデバッグログの有効化
/fewer-permission-prompts権限プロンプトを減らす設定
プラグインで追加したスキルは /skills で確認できます。MCPサーバーが提供するプロンプトは /mcp__サーバー名__プロンプト名 形式で呼び出せます。
