Claude Code クイックスタート

インストール

Node.js 18+ が必要
npm install -g @anthropic-ai/claude-code
バージョン確認
claude --version

起動

プロジェクトディレクトリで実行
cd ~/projects/my-app
claude
プロジェクトの CLAUDE.md が自動読み込みされる

基本操作

操作方法
質問するそのままテキストを入力
ファイルを読む「〇〇.mdを読んで」と指示
コードを修正「〇〇のバグを修正して」と指示
終了/exit または Ctrl+C
ヘルプ/help

便利なスラッシュコマンド

コマンド説明
/helpヘルプ表示
/exit終了
/clear会話履歴をクリア
/compact会話を要約してコンテキスト節約
/cost現在のセッションのトークン使用量
/modelモデル切り替え
/permissions権限設定の確認

CLAUDE.md（プロジェクト設定）

プロジェクトルートに配置
CLAUDE.md
コーディング規約、使用技術、禁止事項などを書いておくと自動で守ってくれる
書き方の例
# プロジェクト名
- 言語: Python 3.12
- テスト: pytest
- コミットメッセージは日本語で
- rm -rf は禁止

権限モード

モード説明
default危険操作のみ確認
plan実行前に計画を提示
bypassPermissions全自動（確認なし）
settings.json の defaultMode で設定
