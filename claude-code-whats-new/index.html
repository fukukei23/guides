🚀 v2.1.154 — 2026-05-14（最大のアップデート）

⚠️ Dynamic Workflows のみ ❌ 利用不可。他機能（Opus 4.8、fast mode）はGLM経由では不可

Dynamic Workflows — 概要

Claude が JavaScript スクリプトを自動生成し、数十〜数百のサブエージェントを並列実行する仕組み。

いつ使う？ コードベース全体のバグ監査・数百ファイルのマイグレーション・複数角度からの検証
いつ使わない？ 2〜5個程度の通常委譲（オーバーヘッド过大）

Ultracode設定

/effort ultracode ← Dynamic Workflows 有効化
/effort high ← 元に戻す

注意点

⚠️ トークン消費が通常の5〜10倍。routine 作業には向かない。
⚠️ Anthropic API（Sonnet/Opus）必需。GLM-5.1環境では動かない。

Opus 4.8 関連（GLMでは不可）

機能説明
