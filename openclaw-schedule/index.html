OpenClaw スケジュール一覧

OpenClawでは、複数のタスクが自動でスケジュール実行されています。
このページでは、各タスクの実行時刻・内容・確認方法をまとめています。

AIニュース収集（07:00 / 19:00）

何をする？
AI関連のニュースをRSSやAPIから収集し、重複を除去して重要度を判定。各記事を日本語で要約し、Discordチャンネルにフォーマット済みで投稿します。

実行確認方法：

  • Discordのニュース投稿チャンネルに新しいメッセージがあるか確認
  • ログに「news」関連の実行記録が残っているか確認
docker compose logs news-collector --tail=20

手動実行：

docker compose exec news-collector npm run collect

動画要約（20:00）

何をする？
指定された動画（YouTubeなど）の内容をテキスト化し、AIで要約を生成。Discordに要約結果を投稿します。ショート動画の自動生成も行う場合があります。

実行確認方法：

  • Discordの動画要約チャンネルに投稿があるか確認
  • ログに実行記録があるか確認
docker compose logs video-summarizer --tail=20

手動実行：

docker compose exec video-summarizer npm run summarize

バックアップ（04:00）

何をする？
OpenClawの設定ファイル、データベース、ログなどをアーカイブして安全な場所に保存します。万が一のデータ消失時に復旧できます。

実行確認方法：

  • バックアップディレクトリに最新のファイルがあるか確認
  • ログにバックアップ完了の記録があるか確認
docker compose logs --tail=30 | grep -i backup

結果の確認先：バックアップ保存ディレクトリ（環境依存）

ヘルスチェック（6時間ごと）

何をする？
各コンテナが正常に動いているか自動確認。問題があればDiscordにアラートを通知します。

実行確認方法：

  • 問題がない場合、通知は来ない（＝正常）
  • Discordにアラートメッセージが来た場合は要対応

手動確認：

docker compose ps

SSOT自動コミット（5分ごと）

何をする？
知識ベース（Obsidian SSOT）の変更を検知し、自動でGitにコミット・プッシュします。変更が失われるのを防ぎます。

実行確認方法：

  • Gitのコミット履歴に自動コミットが追加されているか確認
git log --oneline -10
