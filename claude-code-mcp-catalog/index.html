MCPとは

MCP（Model Context Protocol）は、Claude Codeから外部サービス（GitHub、ブラウザ、検索エンジン等）を直接操作するための標準プロトコルです。

「万能リモコン」のようなもの — 別々のサービスを1つのインターフェースで操作できます。

設定方法の詳細は Claude Code MCP設定ガイド を参照。

追加方法

~/.claude/settings.json の mcpServers に追加:

{
  "mcpServers": {
    "brave-search": {
      "command": "npx",
      "args": ["-y", "brave-search-mcp-server"],
      "env": { "BRAVE_API_KEY": "你的APIキー" }
    }
  }
}

または /mcp コマンドで対話的に追加できます。

APIキーは ~/.secrets.env に一元管理し、設定ファイルには変数名のみ書くのが安全です。
