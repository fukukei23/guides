Tailscale VPN セットアップガイド

Tailscale とは

無料のVPNサービス。デバイス同士をプライベートネットワーク（tailnet）で繋ぐ。設定不要で、インストール＋ログインだけで使える。

自宅PCと外出先のスマホを繋いで、どこからでも自宅PCにアクセス可能にする

PC（Windows）へのインストール

1. インストール
winget install Tailscale.Tailscale
2. 起動してログイン（Google/GitHub/Microsoft アカウント）
3. タスクバーのアイコンから接続確認

スマホ（Android）へのインストール

1. Play Store で「Tailscale」をインストール
2. PCと同じアカウントでログイン
3. VPN接続をオンにする
PCと同じアカウントを使うことが重要。違うアカウントだと繋がらない

接続確認

自分のIP確認（PC側）
tailscale ip -4
スマホからPCにping
ping 100.xxx.xxx.xxx
スマホからPCにSSH接続
ssh ユーザー名@100.xxx.xxx.xxx

管理コンソール

https://login.tailscale.com/admin/machines でデバイス一覧・接続状態を確認
確認項目場所
デバイス一覧Machines
接続状態各デバイスの緑/灰色アイコン
IPアドレス確認各デバイスの詳細
