Obsidian使い方

Obsidianとは

Markdownベースのメモアプリ。リンクで知識を繋げる「セカンドブレイン」ツール。

全データがローカルのMarkdownファイル。ベンダーロックインなし。

Vault（保管庫）

Obsidianのデータは「Vault」というフォルダに全て保存される。

my-vault/
├ 00_System/      ← 体系・ルール
├ 01_Decisions/   ← 決定記録
├ 10_Daily/       ← 日記
├ 20_Notes/       ← ノート
└ 99_Archive/     ← アーカイブ

Wikilink（リンク）

[[ファイル名]]               ← リンク
[[ファイル名|表示名]]         ← 別名で表示
[[ファイル名#見出し]]         ← 見出しへジャンプ
![[ファイル名]]               ← 埋め込み表示
設定 → ファイルとリンク → 「新しいリンクの形式」を「最短パス」に推奨

基本操作

操作方法
新規ノートCtrl+N
クイックスイッチャーCtrl+O（ファイル名で検索）
検索Ctrl+Shift+F（全文検索）
コマンドパレットCtrl+P（全コマンド実行）
グラフビューCtrl+G（リンクの繋がりを可視化）
バックリンク右サイドバー or Ctrl+Shift+B

タグ

#タグ名        ← インラインタグ
#親/子          ← 階層タグ
タグペインで一覧・フィルタリング可能

おすすめコミュニティプラグイン

プラグイン用途
Templaterテンプレート（日付変数等）
DataviewSQLライクにノート検索・集計
Calendarカレンダー表示・デイリーノート
GitGit自動バックアップ
LinterMarkdownの自動整形

Git同期（推奨）

Obsidian Git プラグインで自動コミット＆プッシュ:
1. VaultをGitHubリポジトリにする
2. Obsidian Git プラグインをインストール
3. 自動バックアップ間隔を設定（例: 30分）
複数PC間でも Git で同期可能
