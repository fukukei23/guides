VS Code + WSL2

概要

VS CodeからWSL2のLinux環境を直接使える拡張機能。Windows上のVS CodeでLinuxのコードを編集・実行できる。

セットアップ

1. VS Codeに拡張機能をインストール:
code --install-extension ms-vscode-remote.remote-wsl
2. WSLターミナルからVS Codeを起動:
cd ~/projects/my-app && code .
左下に「WSL: Ubuntu」と表示されれば接続成功

推奨拡張機能

拡張機能用途
PythonPython開発（必須）
Pylance型チェック・補完
GitLensGit履歴の表示
ESLintJavaScript/TypeScript Linter
Prettierコードフォーマッター
Thunder ClientAPI テスト（Postman代替）
Error Lensエラーをインライン表示
拡張機能は「WSL側」と「Windows側」それぞれにインストールが必要。WSL接続時にインストール推奨。

ターミナル統合

VS Code内でWSLターミナルが使える:
Ctrl+` でターミナルを開く（自動的にWSLシェル）
複数ターミナル: ターミナル → 新しいターミナル

「WSL接続中」のままWSLが起動しているか確認: wsl
